Un sistema de inventario el conjunto de normas, métodos y procedimientos que se utiliza para planificar y controlar los productos o materiales que utiliza una empresa, de manera que esta pueda funcionar eficazmente.

El sistema de inventario es un proceso básico para todo tipo de empresas y, en especial, para aquellas que gestionan un número muy variado de productos. Conocer la localización exacta de las referencias evita errores y retrasos en las tareas de almacenaje y preparación de pedidos.

Con un sistema de inventario adecuado, las compañías pueden calcular la cantidad de productos que necesitan para prestar servicio, asegurar la distribución de la mercancía en los plazos acordados e, incluso, hacer frente a cambios imprevistos en la demanda.
